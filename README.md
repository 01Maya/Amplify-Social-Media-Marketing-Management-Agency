# 💜 Amplify – Social Media Agency Landing Page 🪄

A sleek, responsive, and animated digital marketing website built to help brands grow through storytelling, strategy, and analytics.

## 🪶 Overview
Amplify is a modern, motion-driven landing page for digital agencies — designed to showcase services, pricing, and testimonials with smooth transitions and clean UI.

## 📝 Description
This landing page blends creativity and precision with seamless animations, flipping service cards, and an interactive logo carousel.  
It’s fully responsive and optimized for performance, providing users a delightful browsing experience.  
From scroll-triggered motion to intuitive navigation, Amplify redefines how agencies present their digital presence online.

---

## ✨ Features
- **Responsive Design** – Fully optimized for desktop, tablet, and mobile.  
- **Slide-In/Up Animations** – Smooth section transitions using Framer Motion.  
- **Ease-In-Out Effects** – Natural and fluid animations for scrolling and interactions.  
- **Hovering Effects** – Engaging hover states for buttons, cards, and icons.  
- **Logo Carousel** – Auto-sliding client/brand logos with infinite loop.  
- **Flip Card Effect** – Service cards reveal details on hover or tap.  
- **Scroll Navigation** – Smooth scroll-to-section navigation for easy browsing.  
- **Pricing Toggle** – Monthly and yearly plan switch with subtle animation.  
- **Testimonials Slider** – Slide-based client feedback display.  .  
- **Contact Form** – Functional email and message form with clean UI.  

---

## 📂 Sections

- **Hero:** Captures attention with tagline, CTAs, and trusted brand logos carousel.  
- **Who We Are:** Introduces the agency, stats, and achievements.  
- **What We Do:** Displays services using flip-card animations.  
- **Pricing:** Interactive pricing section with monthly/yearly toggle.  
- **Testimonials:** Sliding client reviews showcasing success stories.  
- **Contact:** Lets users reach out via form, email, or live chat.  
- **Footer:** Contains quick links, resources, and social icons.

---

## 🧩 Tech Stack
- **Next.js** – React framework for fast and SEO-optimized web apps  
- **Tailwind CSS** – For responsive, utility-first styling  
- **Framer Motion** – Powering smooth animations and transitions  
- **Shadcn UI** – Modern and accessible UI components
